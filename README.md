This is a [Next.js](https://nextjs.org/)+[Antd](https://ant.design/) mini project.

## Getting Started

```bash
# intall dependencies
npm i
# or
yarn

# develop
npm run dev
# or
yarn dev

# release
npm run build
# or
yarn build
```
